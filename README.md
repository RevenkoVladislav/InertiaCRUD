### Install and settings

```
git clone https://github.com/RevenkoVladislav/InertiaCRUD.git
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
npm install
npm vite
php artisan serve
php artisan schedule:work
```
---

### Features:

- Full CRUD functionality for posts.
- Real-time validation error handling.
- Seamless SPA experience using Laravel and Inertia.
